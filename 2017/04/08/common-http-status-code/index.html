當用戶向伺服器發出請求時，狀態碼能夠描述伺服器返回的結果。

  1. 2XX 成功

    1. 200 OK
    2. 204 No Content
    3. 206 Partial Content

  2. 3XX 重新導向

    1. 301 Moved Permanently
    2. 302 Found
    3. 303 See Other
    4. 304 Not Modified
    5. 307 Temporary Redirect

  3. 4XX 客戶端錯誤

    1. 400 Bad Request

      伺服器無法理解當前請求

    2. 401 Unauthorized
    3. 403 Forbidden
    4. 404 Not Found

  4. 5XX 伺服器錯誤

    1. 500 Internal Server Error
    2. 503 Service Unavailable
